## Benefits of AI

- Helped generate an initial layout and structure quickly.
- Provided ideas for features such as theme toggle and project filters.
- Assisted in drafting documentation, which I then edited and personalized.

## Challenges

- AI sometimes produced more code than I needed; I had to simplify.
- I needed to carefully read and understand all code to avoid relying on it blindly.
- Ensuring that the result matched course requirements and my own style.

## Learning Outcomes

- Better understanding of responsive layouts using CSS Grid and Flexbox.
- Learned how to implement dark/light theme toggling with `localStorage`.
- Improved my ability to review, adapt, and refactor AI-generated code.
- Practiced transparent and ethical reporting of AI usage.
